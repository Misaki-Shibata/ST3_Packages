# ST3_Packages
SublimeText3 Packages 同期用リポジトリ

windowsの場合

```
$env:appdata\Sublime Text 3\Packages\
```

mac(OSX)の場合
```
~/Library/Application Support/Sublime Text 3/Packages/
```

###使い方メモ

* テーマ変更

ツール->Command Palette「ColorSchemeSelector: Select Color Scheme」を入力後十字キー↑↓ 

* Mac用の設定 

iTerm2の起動
